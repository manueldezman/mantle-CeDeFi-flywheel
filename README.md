## The CeDeFi Flywheel

The flywheel has 5 interconnected components that feed each other in a self-reinforcing loop:

### 1. mETH Protocol (Liquid Staking)
Users stake ETH → receive mETH (liquid staking token) → earns staking yield while staying liquid.

### 2. Mantle Treasury
The DAO treasury holds large ETH/mETH reserves → deployed across DeFi/CeFi to generate yield → funds ecosystem grants & incentives.

### 3. Mantle Network (L2)
Low-cost EVM L2 → attracts DeFi protocols → creates demand for mETH as collateral → transaction fees flow back to treasury.

### 4. CeDeFi Yield Products
Institutional-grade yield (e.g. FBTC, COOK) → bridges CeFi liquidity into Mantle's DeFi ecosystem → higher TVL → more attractive L2.

### 5. Ecosystem Grants & Incentives
Treasury funds builder grants → new protocols on Mantle → more users → more staking demand → mETH supply grows.

## The Loop

mETH staking → generates yield → grows Treasury → funds Grants → attracts protocols to Mantle L2 → protocols use mETH as collateral via CeDeFi products → more users stake → back to mETH staking ♾️
